# Task Management App Aqeeq Technologies

## Prerequisites

- React
- Next.js
- Mongodb (mongoose)
- Prisma
- Clerk ( for auth )

## Installing

**1. Clone the repository**

- run `git clone "url"`

**2. Install dependencies**

- run `npm install`

**3. Run the server**

- run `npm run dev`

## Environment Variables

DATABASE_URL

NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY

CLERK_SECRET_KEY
